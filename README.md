# Doraemon UI

A React-based frontend for the Doaraemon project. This application includes various features such as dashboard with widgets, to-do list, settings, and a chatbot powered by a language model Api.

## ?? Features

- **User Authentication** (login, register, reset password)
- **Dashboard** with customizable widgets and statistics
- **To-Do List** with filters, add/edit tasks, reminders
- **Settings/Profile** management
- **Interactive ChatBot** using LLM APIs
- **Responsive design** for desktop and mobile
